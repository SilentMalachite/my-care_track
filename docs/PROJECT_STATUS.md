## 💡 技術的決定事項

### 採用技術の理由
1. **SQLite3**: オフライン動作、シンプルな運用
2. **Electron**: クロスプラットフォーム対応
3. **React**: コンポーネント再利用性
4. **TypeScript**: 型安全性
5. **TailwindCSS**: 迅速なUI開発

### アーキテクチャ決定
1. **モノレポ構成**: 開発効率を重視
2. **REST API**: シンプルで理解しやすい
3. **JWT認証**: ステートレスな認証
